Introduction

This is a SoundFont2 parser (and soon to be editor), written in TypeScript. It works in Node.js and (modern) web browsers. The parser is based on the SoundFont 2.01 specification, but is currently not fully compliant yet. The end goal is to be fully compliant with the SoundFont 2.04 specification.

This library is not ready for production yet, hence the version 0.x.x. Some SoundFonts may be parsed incorrectly and the API may have breaking changes in the future. The first release that is suitable for production use-cases will be version 1.0.0 and it will follow semver from there on.
